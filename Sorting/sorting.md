# Sorting Algorithms

```javascript
const fib_r = (index_value) => {
  if (index_value === 0) {
    return 0;
  }
  if (index_value === 1) {
    return 1;
  }
  return fib_r(index_value - 1) + fib_r(index_value - 2);
};
```
